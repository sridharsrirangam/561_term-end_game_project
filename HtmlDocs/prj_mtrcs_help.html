Project Metrics

Volume Metrics

By Lines
Total Lines : Sum of number of lines in all files of the project
Number of blank lines : Sum of blank lines in all files of the project
Number of inactive lines : Sum of conditioned out lines in all files of the project
Number of comment lines : Sum of comment lines in all files of the project
Number of commented Source Lines : Sum of lines of code with comments in all files of the project
Number of code lines : Code lines are lines of executable code - i.e. lines excluding the left and the right braces. For a project, this is the summation of code lines in all files of the project.
Number of inactive code lines : Number of code lines that are inactive - for all files in the project
Number of declaration lines : Number of code lines containing declarations - for all files in the project
Number of statement lines : Number of code lines containing only statements - for all files in the project

 

By Code Flow
Total Statements : Total number of statements of code - for all files in the project
Number of conditional statements : Total number of "if" statements in a project
Number of loop statements : Sum of loop statements - for, while, do-while in all files in the project
Number of switch statements : Sum of switch statements - in all files of the projects
Number of case labels : Sum of case labels - in all files in the project
Number of labels : Sum all labels - in all files of the project
Number of jump statements : Sum of all "break", "continue" and "goto" statements in all files of the project
Number of return statements : Sum of all "return" statements in all files of the project.
Maximum depth : The maximum indent level in a function for all files in the project

 

By Declarations and Preprocessor Directives
Total declarations : Number of objects defined or declared in a project - in the global as well as local scopes
Number of initialied declarations : In a project, this denotes the number of objects that have been initialized while being declared
Number of unused declarations : In a project, this denotes the objects that have been declared but have not been used.
Total Preprocessor Directives : Number of #define and #undef directives in a project
Total Conditional Directives : Number of #if, #ifdef, etc type of conditional compilation directives in a project

 

By Identifiers
Total Global Variables : Number of globals defined in the project
Total Functions : Number of functions defined in the project
Total Macros : Number of macros - #defines in the project
Total Types : Number ot types defined in the project
Total Structs : Number of structures defined in the project
Total Unions : Number of unions defined in the project
Total Enums : Number of enums defined in the project
Number of Global Variables used : Number of globals used in the project - these include the project globals and non project or library globals
Number of Functions used : Number of functions used in the project - including the library functions
Number of Macros used : Number of macros used in the project - including the library macros
Number of Types used : Number of types used in the project - including the library types
Number of structs used : Number of structures used in the project - including the library structs
Number of unions used : Number of unions used in the project - including the library unions
Number of enums used : Number of enums used in the project - including the library enums
