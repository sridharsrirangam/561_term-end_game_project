File Metrics

By Volume : Lines

Full File
Total : Number of lines in a file
Blank : Number of blank lines in a file
Inactive : Number of conditioned out lines in a file
Comment : Number of comment lines in a file
COD & CMT : Number of lines of code with comments in a file

 

Code Only
Total : Code lines are lines of executable code - i.e. lines excluding the left and the right braces. For a file, this is equal to all such lines in the file.
Inactive : Number of Inactive Code lines in a file
Declarations : Number of declaration lines in a file
Statements : Number of statement lines in a file

 

By Volume : Control Flow
 

Statements
Total : This is the total number of all statements in a file
Condn : Number of if statements in a file
Loop : Number of loop statements - while, for, do-while in a file
Switch : Number of switch statements in a file
Case : Number of case statements in a file
Label : Number of labels in a file
Jump : Number of break, continue or goto statements in a file
Return : Number of return statements in a file

 

Level

   The maximum indent level of a function in a file

 

By Volume : Declarations

Declarations
Total : Number of objects declared or defined in a file
Initialized : Number of declared objects initialized in a file
Unused : In a file, the number of objcts declared but not initialized

 

Preprocecessor
Includes : The number of include directives in a file
PPDs : The number of #define or #undefs in a file
CCDs : The number of conditional compilation directives in a file

 

By Volume : Identifiers

Defined
vars : Number of global variables declared or defined in a file
funcs : Number of functions defined in a file
macros : Number of #defines in a file
types : Number of types defined in a file
structs : Number of structs defined in a file
unions : Number of unions defined in a file
enums : Number of enums defined in a file

 

Used
vars : Number of global variables used in a file
funcs : Number of functions used in a file
macros : Number of #defines used in a file
types : Number of types used in a file
structs : Number of structs used in a file
unions : Number of unions used in a file
enums : Number of unions used in a file

 

By Complexity : Halsteads
 

Length  N
    The program length (N) is the sum of the total number of operators and operands in the program - N = N1 + N2
Vocabulary  n
    The vocabulary size (n) is the sum of the number of unique operators and operands - n = n1 + n2
Volume  V
    The vocabulary size (n) is the sum of the number of unique operators and operands - n = n1 + n2
Difficulty  D
    The vocabulary size (n) is the sum of the number of unique operators and operands - n = n1 + n2
Effort  E
    The effort to implement (E) or understand a program is proportional to the volume and to the difficulty level of the program - E = V * D
Bugs Delivered  B
    The number of delivered bugs (B) correlates with the overall complexity of the software. Halstead gives the following formula for B. B = ( E ** (2/3) ) / 3000 where ** is "to the exponent"
