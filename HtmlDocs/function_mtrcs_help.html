Function Metrics

By Volume : Lines

Full Function
Total : Number of lines in a function
Blank : Number of blank lines in a function
Inactive : Number of conditioned out lines in a function
Comment : Number of comment lines in a function
COD & CMT : Number of lines of code with comments in a function

 

Code Only
Total : Code lines are lines of executable code - i.e. lines excluding the left and the right braces. For a function, this is equal to all such lines in the range (i.e between the function start line and the function end line) of the function.
Inactive : Number of Inactive Code lines in a function
Declarations : Number of declaration lines in a function
Statements : Number of statement lines in a function

 

By Volume : Control Flow

Recursive

   It's values are No - if the function is not recursive, Directly - if the function calls itself and Indirectly - if the function is recursive - not directly, but through some other sequence instead.
 

Statements

Total : This is the total number of all statements in a function
Condn : Number of if statements in a function
Loop : Number of loop statements - while, for, do-while in a function
Switch : Number of switch statements in a function
Case : Number of case statements in a function
Label : Number of labels in a function
Jump : Number of break, continue or goto statements in a function
Return : Number of return statements in a function

 

Level

   The maximum indent level of a function
 

Height

   The maximum level upto which functions are called by a function - i.e the maximum distance between the root and any of the leaves in a function call tree
 

Weight

   The number of unique functions called by a function - directly or indirectly

 

By Volume : Declarations

Declarations
Total : Number of objects declared or defined in a function
Initialized : Number of declared objects initialized in a function
Unused : In a function, the number of objects declared but not initialized

 

Preprocecessor
Includes : The number of include directives in a function
PPDs : The number of #define or #undefs in a function
CCDs : The number of conditional compilation directives in a function

 

By Volume : Identifiers

Defined
vars : Number of global variables declared or defined in a function - NOT APPLICABLE - AS NOT POSSIBLE TO DECLARE/DEFINES GLOBALS IN A FUNCTION
funcs : Number of functions defined in a function - NOT APPLICABLE - AS NOT POSSIBLE TO DEFINE FUNCTIONS WITHIN ANOTHER FUNCTION IN C
macros : Number of #defines in a function
types : Number of types defined in a function
structs : Number of structs defined in a function
unions : Number of unions defined in a function
enums : Number of enums defined in a function

 

Used

vars : Number of global variables used in a function
funcs : Number of functions used in a function
macros : Number of #defines used in a function
types : Number of types used in a function
structs : Number of structs used in a function
unions : Number of unions used in a function
enums : Number of enums used in a function
